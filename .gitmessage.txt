# --- COMMIT END ---
#   <타입> 리스트
INIT     : 프로젝트 초기 설정 작업
DEPS     : 의존성 변경 (다운/업그레이드, 추가/삭제, 고정 등)
CONFIG   : 설정 파일 추가/업데이트, .gitignore 수정 등
FEAT     : 새로운 기능 추가
DEL      : 코드/ 파일 삭제
MOVE     : 파일, 경로, 라우트 등 리소스 이동/이름변경
REFACTOR : 코드 리펙토링, 기능 변화 없이 구조 수정
PERF     : 성능 개선
SECURITY : 보안 이슈 수정, 암호화 추가 또는 업데이트
FIX      : 버그 수정, 컴파일러 / 린터 경고 수정
HOTFIX   : 배포 및 이슈로 인해 긴급하게 긴급 수정
CHORE    : 개발 스크립트 추가 및 수정, 코드/외부 API 관련 업데이트
DB       : 데이터베이스 관련 변경 수행
STYLE    : UI 스타일링 추가/업데이트, 반응형 디자인, 애니메이션 전환
REVERT   : 변경사항 되돌리기
MERGE    : 브랜치 병합(머지)
TEST     : 테스트 추가/업데이트 또는 테스트 통과
CI       : CI 빌드 수정, CI 빌드 시스템 추가 또는 업데이트
A11Y     : 접근성 개선
SEO      : 검색 엔진 최적화 개선
DOCS     : 문서 추가/업데이트, 라이센스 추가/업데이트
RELEASE  : 배포, 릴리즈 / 버전 태그
LOG      : 로그 추가/업데이트, 로그 제거
COMM     : 소스 코드 내 주석 추가/업데이트

#   Resolves     : 이슈 해결했을 때 사용  
#   Ref          : 참고할 이슈가 있을 때 사용  
#   Related to   : 해당 커밋에 관련된 이슈번호 (아직 해결되지 않은 경우)  
#   ex) Fixes: #47 Related to: #32, #21
